# 🚀 CompanyGPT  
### AI Operating System for Modern Teams

> CompanyGPT is an AI-powered SaaS workspace that transforms meetings, tasks, documents, analytics, and company workflows into an intelligent automated operating system.

Built for startups, remote teams, and modern organizations using **React**, **Next.js**, **Firebase**, and **Gemini AI**.

---

# 🌟 Overview

Managing company operations across multiple tools is inefficient and fragmented.

CompanyGPT solves this by combining:
- AI-powered reporting
- intelligent meeting analysis
- automated task extraction
- productivity analytics
- AI chat assistance
- WhatsApp operational alerts

into a single unified platform.

# ⚡ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/companygpt.git
```

---

## 2️⃣ Navigate to Project

```bash
cd companygpt
```

---

## 3️⃣ Install Dependencies

```bash
npm install
```

---

## 4️⃣ Setup Environment Variables

Create `.env.local`

```env
NEXT_PUBLIC_FIREBASE_API_KEY=
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=
NEXT_PUBLIC_FIREBASE_PROJECT_ID=
NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=
NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=
NEXT_PUBLIC_FIREBASE_APP_ID=

GEMINI_API_KEY=

TWILIO_ACCOUNT_SID=
TWILIO_AUTH_TOKEN=
TWILIO_WHATSAPP_NUMBER=
```

---

# 🚀 Run Locally

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```
